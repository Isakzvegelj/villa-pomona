# Clawd Project

This project contains Ralph, an autonomous background daemon, and Clawd, an AI familiar.

## Ralph

Ralph is the persistent background layer that handles Discord communication and task monitoring.

**Current Version:** v1.1.0

### Version Check Commands
- **Discord:** Type `!version` or `!status` to see the current version.
- **CLI:** Run `python ralph_daemon.py --version` from the `clawd` directory.

## Clawd

Clawd is the AI personality that processes tasks from `INBOX.md`.
