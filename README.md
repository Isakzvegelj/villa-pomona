# Villa Pomona — Bled, Slovenia

A historic boutique villa website featuring elegant rooms, wellness & spa, and gourmet breakfast in the heart of Bled, Slovenia.

## Features

- 🎨 Modern, responsive design with smooth animations
- 🌿 Green-themed branding distinct from Villa Adora
- 💬 Built-in virtual concierge chat widget
- 🖼️ Photo gallery with lightbox
- 📝 Integrated reservation form
- 🌐 Bilingual support (EN/SL)
- 📱 Mobile-first responsive design
- 🔍 SEO-optimized with JSON-LD structured data
- 🗺️ Google Maps integration
- ⭐ Guest testimonials carousel
- 🏆 Awards & ratings section

## Tech Stack

- Static HTML/CSS/JS — no build step required
- Vanilla JavaScript (no dependencies)
- Font Awesome 6 icons
- Google Fonts (Playfair Display + Inter)
- CSS custom properties for theming

## Pages/Sections

- Hero with auto-sliding image carousel
- About / Our Story
- Rooms & Suites (Standard, Deluxe, Family)
- Wellness & Spa
- Photo Gallery with lightbox
- Guest Testimonials
- Reservation Form
- Contact & Location with Google Maps
- Footer with quick links and social media

## Deployment

Can be deployed to any static host (GitHub Pages, Netlify, Vercel, Render).
